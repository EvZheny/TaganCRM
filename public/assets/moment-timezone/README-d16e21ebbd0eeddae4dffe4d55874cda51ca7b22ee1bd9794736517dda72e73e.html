Moment Timezone

IANA Time Zone Database + Moment.js.

```js var june = moment("2014-06-01T12:00:00Z"); june.tz('America/LosAngeles').format('ha z'); // 5am PDT june.tz('America/NewYork').format('ha z'); // 8am EDT june.tz('Asia/Tokyo').format('ha z'); // 9pm JST june.tz('Australia/Sydney').format('ha z'); // 10pm EST

var dec = moment("2014-12-01T12:00:00Z"); dec.tz('America/LosAngeles').format('ha z'); // 4am PST dec.tz('America/NewYork').format('ha z'); // 7am EST dec.tz('Asia/Tokyo').format('ha z'); // 9pm JST dec.tz('Australia/Sydney').format('ha z'); // 11pm EST ```
